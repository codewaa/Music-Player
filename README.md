# Chapter 5 JavaScript Mini Project

## Project Name

Mini Music Player

## Concepts Practiced

- HTML forms
- HTML audio player
- CSS grid layout
- Modern card UI
- JavaScript arrays and objects
- Functions
- DOM selection
- Events
- Free music API
- Fallback data

## Student Task

Build a small music player. The user can search music, view song cards, click Play, and listen using the audio player.

The project uses the free iTunes Search API:

`https://itunes.apple.com/search?term=happy&media=music&entity=song&limit=12`

The API returns real song preview audio URLs and artwork. If the API is not available, JavaScript shows built-in playable demo tracks so the project still works in class.

## Challenge Ideas

1. Add one new demo track in `script.js`.
2. Change the color of the now-playing cover.
3. Add a favorite button.
4. Show the current song duration.
5. Add a clear search button.

## How To Run

Open `index.html` in a browser.
